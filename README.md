# AI Demand Collapse and the Bubble-Crash Extension of the Imas Model

**Author:** Guy-Philippe Goldstein
**Date:** April 2026
**Status:** Working paper — first draft

This document extends the Keynesian demand-determined equilibrium model
developed by Alex Imas ("Can Advanced AI Lead to Negative Economic Growth?",
Ghosts of Electricity, January 2026) with a financial intermediation block
connecting distributional dynamics to asset bubble formation.
